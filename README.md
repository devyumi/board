## `프로젝트 소개`

글과 댓글을 작성할 수 있는 게시판

로그인 기능이 없는 대신 **게시글(댓글) 등록 시 사용했던 비밀번호를 이용**하여 게시글 수정 및 삭제를 할 수 있다.
